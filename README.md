This project is a Linear Regression web application built using Streamlit and Scikit-learn. It enables users to easily train and evaluate a regression model on their own dataset.

🔧 Features
Upload and preview dataset

Select target variable

Automatic preprocessing:

Handles missing values

Encodes categorical data

Converts date columns to numeric

Train Linear Regression model

Evaluate using:

Mean Squared Error (MSE)

Root Mean Squared Error (RMSE)

Mean Absolute Error (MAE)

R² Score

Make real-time predictions through UI

🚀 Tech Stack
Python

Streamlit

Scikit-learn

Pandas / NumPy
